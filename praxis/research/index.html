The Research

No single event type carries the signal. Co-occurrence does.

The original research hypothesis, that kidnapping spikes alone would function as an early warning signal for violent extremism, was falsified. A within-month temporal ordering test found no lead: kidnapping events do not precede violent extremism attacks within spike-months (Wilcoxon p = 0.68). The original finding reflected conflict autocorrelation, not a lead signal.

The expanded analysis across 13 countries and 100,184 events revealed a different structure. When two or more distinct conflict event types converge in the same district-week, that district sits in a small cluster averaging 8.8 times the Sahel-wide clean baseline attack rate (N=292, p < 0.001, exclusive of triggering types). The signal is not carried by any single event type. It emerges from the convergence of multiple armed group capabilities operating simultaneously in the same geography. It identifies where persistent violence concentrates.

Method

Weekly district-level panel across 13 countries (Burkina Faso, Mali, Niger, Nigeria, Chad, Cameroon, Mauritania, Togo, Benin, Ghana, Senegal, Guinea, Côte d'Ivoire), 1,503 admin2 districts, January 2010 to March 2025. ACLED event data classified by sub-event type. Composite signal defined as two or more distinct conflict sub-event types co-occurring in the same district-week. District-week outcome rates compared using Mann-Whitney U tests against a clean baseline of non-composite district-weeks. Exclusive signal tests remove the triggering types (KAFD, IED) from the outcome set to eliminate predictor-outcome overlap.

The composite co-occurrence signal operates on two levels. First, as a district identifier: districts where KAFD and IED co-occur in the same week sit in a persistently high-conflict cluster, averaging 8.8 times the Sahel-wide clean baseline attack rate, exclusive of triggering types. This dimension is robust. Second, as a concurrent intensity indicator: within those already-violent districts, the signal week shows an 18 percent increase in non-triggering violence above the pre-signal corridor. This second dimension is provisional pending a panel regression with district fixed effects and time-varying controls. The district identification metric does not depend on the temporal finding.

Methodology

Composite signal detection.

The methodological foundation is composite signal detection at weekly temporal resolution. The analysis pipeline classifies ACLED events by sub-event type and tests whether combinations of types co-occurring in the same admin2-week identify districts with elevated violence. The exclusive signal test strips out concurrent events and removes the triggering types themselves from the outcome set to isolate whether any individual event type carries a standalone signal. It does not. The finding is emergent, arising from the convergence of multiple armed group capabilities operating in the same geography in the same week.

A note on the predictor-outcome set

The exclusive metrics (8.8x district identification, 1.2x temporal escalation) remove KAFD and IED from the outcome count. This eliminates mechanical double-counting of the triggering events inside the outcome variable and produces the methodologically conservative headline. The inclusive metrics are available in the manuscript appendix.

The Central Sahel was the initial proof of concept. The expanded 13-country analysis confirms the district-level signal, with strongest effects in the Lake Chad Basin and Coastal West Africa. Wherever ACLED operates, roughly 230 countries and territories, is the addressable scope. The Central Sahel is the proof of concept because the data is richest there for the composite co-occurrence pathway. It is not the ceiling.

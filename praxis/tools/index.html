The Challenge

Evaluation is hard. AI alone isn't enough.

General-purpose AI doesn't understand evaluation methodology. PRAXIS bridges that gap.

⚠️

Without PRAXIS

  • Generic AI gives surface-level evaluation advice that wouldn't survive a peer review
  • No awareness of OECD-DAC criteria, validated scales, or FCV-adapted methods
  • Cannot distinguish between a DiD and a regression discontinuity — let alone recommend one
  • Produces cookie-cutter surveys with no psychometric grounding
✨

With PRAXIS

  • Claude becomes a methodologist who asks the right questions before recommending designs
  • 20+ evaluation approaches — experimental, quasi-experimental, theory-based, participatory, realist
  • 16+ validated scales (FCS, PHQ-9, WHODAS, WEAI) embedded and ready to deploy
  • Methods adapted for fragile, conflict-affected, and violent settings

3 Levels
Setup

Up and running in three steps

PRAXIS is a Claude skill — a structured knowledge base that gives Claude domain expertise without fine-tuning.

Download the Skill

Clone or download the PRAXIS repository from GitHub. It's a collection of Markdown reference files — no dependencies, no API keys, no build step.

Add to a Claude Project

Upload the PRAXIS files to a Claude Project's knowledge base. Claude will automatically reference them when you ask evaluation questions.

Start Evaluating

Ask Claude to design an evaluation, build instruments, review a ToR, or recommend methods. PRAXIS handles the rest — with the rigour of a senior evaluator.

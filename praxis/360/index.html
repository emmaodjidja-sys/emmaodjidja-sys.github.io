// THE PIPELINE

What 360° connects

Six stages. One file. Every output feeds the next. Every finding traces back to the question that generated it.

01
Theory of Change
Structured causal pathways with assumptions, evidence ratings, and risk flags. The foundation every subsequent step references.
02
Evaluation Matrix
Questions linked to OECD-DAC criteria, indicators, data sources, and methods. Each row traces back to a ToC pathway.
03
Sampling & Design
Sample size calculation, design selection, and power analysis informed by the evaluation questions and context constraints.
04
Instruments
Data collection tools generated from the matrix. Every question on every form maps to an indicator, which maps to a ToC assumption.
05
Analysis Plan
Statistical and qualitative analysis methods pre-registered against each evaluation question. No post-hoc fishing.
06
Findings to Framework
Results mapped back to the Theory of Change. Assumptions confirmed, contested, or revised. The loop closes.
// THE DIFFERENCE

What breaks today vs. what 360° fixes

Current practice
ToC in PowerPoint, matrix in Excel, instruments in Word, analysis in Stata, report in another Word file
No traceability between stages
Findings disconnected from the questions that generated them
Methodology lives in consultant reports that nobody reads twice
PRAXIS 360°
One .praxis file. Every stage linked. Every output traceable.
Change a ToC assumption and the downstream instruments, indicators, and analysis plan update
Every finding maps back to the evaluation question and ToC pathway that produced it
Methodology becomes structured, portable, and reusable

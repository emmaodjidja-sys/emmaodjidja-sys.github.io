PRAXIS Evaluation Track

Programme Evaluation Infrastructure

I spent years looking for evaluation tools built by someone who had actually run an evaluation in a conflict zone. I could not find them. So I built them.

The problem

Evaluation in fragile and conflict-affected settings demands methodological rigour under conditions that make rigour difficult. The tools available to evaluators were built for offices with reliable internet, not for field visits where connectivity drops and platforms change their pricing. I kept losing access to my own work.

PRAXIS runs entirely in your browser. Everything stays on your machine. No server calls, no accounts, no tracking. If the wifi cuts out in Juba or Ouagadougou, the tools still work. If PRAXIS disappears tomorrow, you still have your files. MIT licensed, permanently free, fully yours.

The ecosystem

01

AI Evaluation Skills

Claude Project knowledge modules. Six core modules plus sector-specific extensions. Over 20 evaluation designs and 16 validated scales, encoded as structured methodology that an AI can apply to your specific context.

02

Practitioner Tools

Six browser-based tools covering the full evaluation lifecycle. Sample size calculation, evaluation design advising, data exploration, Theory of Change building, evaluation matrix construction, and an indicator bank with 298 indicators across 11 sectors. Zero data transmission.

03

PRAXIS 360

The integrated end-to-end evaluation workflow built on the .praxis file format. Theory of Change through to reporting, with every finding traceable back to the evaluation framework. Launching at Glocal Evaluation Week 2026.

Live tools

Sample Size Calculator
Rigorous sample size estimation for 8 evaluation designs. RCT, cluster, stepped-wedge, DiD, LQAS.
Evaluation Design Advisor
Answer 10 context questions, get scored recommendations across 16 designs with trade-offs.
Data Explorer
Upload CSV or XLSX for instant descriptive stats, regression, Likert analysis, and Word export.
Theory of Change Builder
Build structured Theories of Change with assumptions mapping and causal pathway visualisation.
Evaluation Matrix Builder
Link evaluation questions to criteria, indicators, data sources, and methods. Export to Word.
Indicator Bank
298 indicators across 11 sectors. PEPFAR MER codes, validated instruments. CSV export.
